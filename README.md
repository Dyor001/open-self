# Open Self｜开源自我

> 过去可以被追溯，思想可以被理解，现在可以被看见，变化可以被验证。

这是一个持续更新的个人开源项目。

我正在尝试把自己能够公开的部分，以真实、结构化、可追溯的方式长期记录下来。它不是一份静态的个人介绍，也不是为了塑造一个完美人设，而是为人与人之间的理解，以及未来 AI 与我的直接协作，建立一份由我本人维护的可靠资料源。

## 当前开放内容

### [我的思想](thoughts/README.md)

记录我如何理解世界、如何判断问题，以及我的价值观和认知变化。

### [我的个人回忆录](memoir/README.md)

记录塑造我的人生阶段、重要经历与真实感受。

### [我正在做的事情](now/README.md)

持续更新我当前投入的项目、方向、目标与进展。

### [我的做人准则](principles/social.md)

记录我在人际关系中的基本原则、相处方式、边界与红线。

## 项目原则

- **真实**：区分事实、自我理解与个人观点。
- **持续**：允许内容随着经历和认知不断更新。
- **可追溯**：保留变化过程，不假装自己从未改变。
- **有边界**：开源自己，不擅自开源他人的隐私。
- **本人定义**：任何人和 AI 都可以阅读与理解，但只有我能定义和更新我。

## 当前状态

项目处于早期阶段。现有内容只是起点，结构和表达会随着我对自己的理解逐步完善。
